# Setting up the frameworks

### Virtual environment 
- This is highly recommended for downloading the required python modules. If you have downloaded other python modules before, it might cause a compatibilty issue. 

### How to setup python environment
- Creating venv: python3 -m venv ~/.virtualenvs/"name of the venv"
- Running venv: source ~/.virtualenvs/"name of the venv"/bin/activate
- Deactivate command: deactivate 

Every venv is stored ".virtualenvs" folder. The path is:  ~/.virtualenvs/
