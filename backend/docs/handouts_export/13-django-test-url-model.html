Django URL and Model Testing

Url tests

  • python manage.py test courses/tests
# courses/tests/test_urls.py

from django.test import TestCase
from django.urls import resolve
from courses.views import index

class CourseUrlTest(TestCase):
    def test_courses_url_resolve_to_course_index_page(self):
        view = resolve('/courses/')
        self.assertEqual(view.func, index)

Model tests

# courses/tests/test_models.py

from django.test import TestCase
from django.core.exceptions import ValidationError
from courses.models import Department, Course

class CourseModelTest(TestCase):
    def test_duplicate_titles_invalid(self):
        cs = Department.objects.create()
        Course.objects.create(dept=cs, title='twice')
        same_course = Course(dept=cs, title='twice')
        with self.assertRaises(ValidationError):
            same_course.validate_unique()
  • python manage.py test courses/tests

For each test case:

  • set up: creating test database …
  • run the test case
  • shut down: destroying test database …

Testing uses a separate test database. When using SQLite (the default), the test database is in-memory.

The are several model validation methods (clean_fields, clean, validate_unique, validate_constraints, full_clean, or ModelForm.is_valid).

Alternatively: 

# courses/tests/test_models.py

from django.test import TestCase
from django.db.utils import IntegrityError
from courses.models import Department, Course

class CourseModelTest(TestCase):
    def test_duplicate_titles_invalid(self):
        cs = Department.objects.create()
        Course.objects.create(dept=cs, title='twice')
        same_course = Course(dept=cs, title='twice')
        with self.assertRaises(IntegrityError):
            same_course.save()

Use database integrity constraints or Django model validation?

Model valiation is cheaper when it does not require a database operation (such as validating string-length constraint, but not for this example, uniqueness). You may also do advanced validation in the model's clear() method.
